Política de Privacidade

Projeto de Vida · Última atualização: maio de 2026

1. Sobre o aplicativo

O Projeto de Vida é um aplicativo gratuito de orientação vocacional para jovens. Esta política explica como tratamos suas informações.

2. Dados armazenados no seu dispositivo

O aplicativo mantém localmente no celular: resultado do teste DISC, resultado do teste Holland, preferências de notificação e progresso no onboarding. Estas informações nunca são transmitidas aos servidores.

3. Anúncios — Google AdMob

Utilizamos o Google AdMob para exibir anúncios. O AdMob coleta identificador do dispositivo e dados de uso para personalização. A desativação está disponível em Configurações → Google → Anúncios (Android).

Referência: https://policies.google.com/privacy

4. Backup de dados

O recurso de backup gera um arquivo exportado diretamente para o seu dispositivo. Não temos acesso a esses arquivos.

5. Crianças

Este aplicativo não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores.

6. Alterações nesta política

Esta política pode ser atualizada periodicamente; a data no topo sempre mostra a versão atual.
